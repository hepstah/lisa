**Create backend/lisa/services/voice_pipeline.py** -- the orchestrator that chains all services.

Define error message constants at module level (per D-17 through D-20):
```python
MSG_STT_TIMEOUT = "Sorry, I could not understand that. Please try again."
MSG_LLM_TIMEOUT = "I'm having trouble processing that right now."
MSG_NO_INTERNET = "Voice understanding is temporarily unavailable."
MSG_UNKNOWN_INTENT = "I didn't understand that. Try saying something like: turn on the bedroom lamp."
MSG_DEVICE_ERROR = "I had trouble controlling that device. Please try again."
MSG_DEVICE_UNREACHABLE = "That device doesn't seem to be reachable right now."
```

Define `class VoicePipeline`:
- `__init__(self, stt: STTService | None, llm: LLMIntentService, tts: TTSService, device_service: DeviceService)`:
  - Store all services. `stt` is None in dev mode (text injection bypasses STT per D-13).
  - Store a logger: `self._log = logging.getLogger("lisa.voice_pipeline")`

- `async def process_text(self, text: str, source: str = "voice") -> dict`:
  - This is the main entry point for both voice (after STT) and text injection (D-13).
  - Step 1: Get device context. Call `states = await self._device_service.get_all_states()`. Convert to list of dicts: `[{"device_id": s.device_id, "alias": s.alias, "is_on": s.is_on} for s in states]`.
  - Step 2: Parse intent via LLM. Wrap in try/except:
    - Call `intent = await self._llm.parse_intent(text, device_context)`
    - Catch `LLMTimeoutError`: call `await self._tts.speak(MSG_LLM_TIMEOUT)`, return `{"status": "error", "error_stage": "llm", "error_message": MSG_LLM_TIMEOUT, "raw_input": text}`
    - Catch `LLMError` (connection/other): call `await self._tts.speak(MSG_NO_INTERNET)`, return `{"status": "error", "error_stage": "llm", "error_message": MSG_NO_INTERNET, "raw_input": text}`
  - Step 3: Handle unknown intent. If `intent is None`: call `await self._tts.speak(MSG_UNKNOWN_INTENT)`, return `{"status": "rejected", "error_stage": "intent", "error_message": MSG_UNKNOWN_INTENT, "raw_input": text}`
  - Step 4: Execute device command. Call `new_state, log = await self._device_service.execute_command(device_id=intent.device_id, action=intent.action, source=source, raw_input=text)`.
  - Step 5: Speak result. If `log.get("status") == "success"`: call `await self._tts.speak(intent.confirmation)`. If `log.get("status") == "rejected"`: call `await self._tts.speak(log.get("error_message", MSG_DEVICE_ERROR))`. If `log.get("status") == "error"` and `log.get("error_stage") == "device_unreachable"`: call `await self._tts.speak(MSG_DEVICE_UNREACHABLE)`. Else: call `await self._tts.speak(MSG_DEVICE_ERROR)`.
  - Step 6: Return the log dict enriched with voice pipeline metadata: `{**log, "tts_spoken": True}`.

- `async def process_audio(self, audio_bytes: bytes) -> dict`:
  - This is the entry point for the full voice path (Pi with microphone).
  - If `self._stt is None`, raise `RuntimeError("STT service not available -- use process_text for dev mode")`
  - Step 1: Transcribe. Wrap in try/except:
    - Call `text = await self._stt.transcribe(audio_bytes)`
    - Catch `STTTimeoutError`: call `await self._tts.speak(MSG_STT_TIMEOUT)`, return `{"status": "error", "error_stage": "stt", "error_message": MSG_STT_TIMEOUT}`
    - Catch `STTError` (connection/other): call `await self._tts.speak(MSG_NO_INTERNET)`, return `{"status": "error", "error_stage": "stt", "error_message": MSG_NO_INTERNET}`
  - Step 2: Call `return await self.process_text(text, source="voice")`
