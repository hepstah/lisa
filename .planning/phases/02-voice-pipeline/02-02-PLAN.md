**Step 3: Create backend/lisa/voice/wake_word.py** per VOICE-01 (wake phrase detection).

This module is Pi-only per D-15. In dev mode it is never instantiated.

Define `class WakeWordDetector`:
- `__init__(self, model_names: list[str] | None = None, threshold: float = 0.5)`:
  - If openwakeword not available, raise `RuntimeError("openwakeword not available -- wake word detection requires Pi environment")`
  - Default `model_names` to `["hey_jarvis"]` if None (dev stand-in per research recommendation; custom "hey_lisa" model is deferred per ADV-03)
  - Call `openwakeword.utils.download_models()` to ensure pre-trained models are available
  - Create `self._model = Model(wakeword_models=model_names)`
  - Store `self._threshold = threshold`
  - Store `self._muted = False` (for TTS echo prevention per pitfall 1)

- `def detect(self, audio_frame: bytes) -> dict[str, float]`:
  - If `self._muted`, return empty dict `{}`
  - Call `prediction = self._model.predict(audio_frame)` -- audio_frame must be 1280 samples (80ms at 16kHz, 16-bit PCM = 2560 bytes)
  - Return dict of model_name -> score for scores above `self._threshold`
  - If no scores above threshold, return empty dict

- `def mute(self)`: Set `self._muted = True` (call before TTS playback per pitfall 1)
- `def unmute(self)`: Set `self._muted = False` (call after TTS playback + cooldown)
- `def is_muted(self) -> bool`: Return `self._muted`

**Step 4: Create backend/lisa/voice/audio_capture.py** per VOICE-02 (utterance capture with VAD).

This module is Pi-only per D-15. In dev mode it is never instantiated.

Define `class AudioCapture`:
- `__init__(self, sample_rate: int = 16000, frame_duration_ms: int = 80, silence_threshold: float = 500.0, max_silence_frames: int = 15, max_capture_seconds: float = 10.0)`:
  - Store all parameters. `silence_threshold` is RMS energy below which a frame is considered silence. `max_silence_frames` is how many consecutive silent frames trigger end-of-speech (15 frames * 80ms = 1.2s of silence). `max_capture_seconds` is a hard cap.
  - NOTE: Do NOT import PyAudio or sounddevice here. Those require actual audio hardware. This class defines the capture logic; the actual audio source is injected.

- `def process_frame(self, frame: bytes) -> bool`:
  - Calculate RMS energy of the frame (interpret as 16-bit signed PCM, compute sqrt of mean of squares)
  - If energy < silence_threshold, increment `self._silence_count`. If energy >= silence_threshold, reset `self._silence_count = 0` and set `self._has_speech = True`
  - Append frame to `self._buffer`
  - Return `True` if capture should continue (not enough silence AND not exceeded max duration), `False` if capture is complete

- `def get_audio(self) -> bytes`:
  - Return concatenation of all buffered frames as a single bytes object

- `def reset(self)`:
  - Clear buffer, reset silence count and has_speech flag

- `def has_speech(self) -> bool`:
  - Return whether any non-silent frame was detected
